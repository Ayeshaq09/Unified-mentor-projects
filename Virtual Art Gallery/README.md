**Virtual Art Galary**

This project display Indian cultural images in a 3D image carousel that contains modern animations and interactive effects.

**Features**

This application contains:
1.3D carousel
2.Indian cultural images
3.Sound effects 
4.Animations
5.Modal to view image details
6.Modal for zoom image functionality

The app is mobile-responsive and is user-friendly.

**Technologies Used**

-HTML5, CSS3, JavaScript, Reactjs
-react-tooltip for tooltips

**Setup Instructions**

Clone the repository
Then, run 'npm start' in vscode terminal to start the app and open the "Available on" links mentioned in terminal after you run the command, 
eg:- (http://localhost:3000)

**How to use the project**
-The application first displays a button to open the image gallery.
-On clicking the button, a 3D image carousel is displayed.

1.The carousel has following animations:
-When it first opens
-It rotates
-On hovering over the image

2. Click the image to open a modal with its discription.
-Opening and closing of the modal has animation.
-Then inside modal, click on the image to zoom it.

3.Inside the zoom-modal click on the image to close it.

Resize the screen to check the responsiveness of the application.
