**Super Mall Web Application**

This project display products and its details and it contains dashboard module, shop module, floor module, offer module and category module.

**Features**

This application contains:
1.Login and Register forms
2.Dashboard
3.Displays all products information 
4.User can view all products
5.User can view details of each product
6.User can compare details of two products
7.Admin can add and modify products, shop, floor, category and offer details.

The app is mobile-responsive and is user-friendly.

**Technologies Used**

-HTML5, CSS3, JavaScript, Reactjs
-Nodejs for backend
-Firebase for data storage
-react-toastify for notifications

**Setup Instructions**

Clone the repository
Run 'cd frontend' in terminal.
Then, run 'npm start both' in vscode terminal to start the app and open the "Available on" links mentioned in terminal after you run the command, 
eg:- (http://localhost:3000)

**How to use the project**

-Once registered the user can login.

The application contains 2 modules:
a.Admin
    1.Products
        Here you can Add Product.
        Manage product details - upload product image, edit, delete.
        View all products - Click on a product to view its details
    2.Shops
        Here you can add, edit, delete the shop details
    3.Categories
        Here you can add, edit, delete the category details
    4.Floors
        Here you can add, edit, delete the floor details
    5.Offers
        Here you can add, edit, delete the offer details

b.User
    1.Products
        The user can view all products
        Click on a product to view its details
        User can select products to compare.
        View products - Category wise, Offer wise, Shop wise and Floor wise.

Resize the screen to check the responsiveness of the application.
