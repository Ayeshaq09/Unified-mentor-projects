Navigate to - https://github.com/Ayeshaq09/Unified-mentor-projects
You will view all 8 projects.

To get the projects onto your system:
  click on "Code" button and copy the link - https://github.com/Ayeshaq09/Unified-mentor-projects.git

Open git bash and run the following command:
  git clone https://github.com/Ayeshaq09/Unified-mentor-projects.git

Once cloning is done, you get all the 8 projects onto your system.

To do respective setup for each project, refer to "Project Setup Instructions" file in each project.
